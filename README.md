# CCBSA Picking Accelerator (Advanced)

## The Vision
This is a **High-Velocity Training Engine** designed to optimize human capital in warehouse operations. By applying supply chain efficiency to the "human nodes" (pickers), this application reduces onboarding time and eliminates SKU-substitution errors at the source.

## Core Features
- **Visual Recognition Module:** Flash-card style training for instant SKU identification (Brand vs. Size).
- **Material Code Mastery:** Rapid-fire drills to memorize 4-digit material codes (e.g., 1040, 5954).
- **Pallet Logic Calculator:** Industrial-grade training for pallet quantity configurations and case counts.
- **Adaptive Timer:** Difficulty scales with user performance to build muscle memory.
- **Industrial UI:** Dark-mode, high-contrast interface designed for warehouse environments.

## Tech Stack
- **React (v18)**
- **Tailwind CSS**
- **State Machine Logic** for game state and scoring.

## How to Run
Simply open `index.html` in any modern web browser. No installation or build steps required—designed for zero-friction deployment on mobile or desktop.
