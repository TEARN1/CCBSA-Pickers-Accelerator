## The Vision
This is a **High-Velocity Training Engine** designed to optimize human capital in warehouse operations. By applying supply chain efficiency to the "human nodes" (pickers), this application reduces onboarding time and eliminates SKU-substitution errors at the source.


## Tech Stack

- **Frontend:** React, Tailwind CSS, JavaScript (ES6+), HTML5, CSS3
- **Libraries:** Font Awesome
- **Tooling:** Babel (for in-browser JSX transpilation)
